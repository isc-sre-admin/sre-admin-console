# Mock up backend

Provide a simple, comprehensive  mock up of the backend based on the contracts 
under ./contracts so that the frontend code can be developed against it and 
so the user interface is able to show realistic data during prototyping, but
so that it is easy to swap out this mock up for actual calls to Lamdba and 
Step Functions state machines. 

Set this up so that making this change takes minimal changes and so it is easy
to switch it back and forth to facilitate independent development of the 
frontend going forward.
