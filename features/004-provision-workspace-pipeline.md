# Enhancement to start pipeline execution dialog screen

To accommodate the additional complexity of inputs for the provision workspace pipelines, this feature will add the ability to have a multi-step flow for the inputs. Specifically, it will allow the end user to start the 
workflow by selecting an individual from the list of AD users. 

They will also need to pick a hardware type (e.g. Standard, Performance, Power, PowerPro). 

They will also need to pick a Root Volume and User Volume size. 

Details on the available options can be found under the relevant AWS documentation:
https://aws.amazon.com/workspaces/desktop-as-a-service/pricing/
