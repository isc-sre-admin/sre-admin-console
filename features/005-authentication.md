# Authenication

Add the necessary user interface elements to support end users signing in. 
The authentication and authorization layer is going to be implemented 
using Amazon Cognito and will be handled in a later feature. For this 
feature, make the sre-admin-console user interface respond to whether
the user has logged in or not. 

Make this as elegant and straightforward as possible, following the
standard practices of how this is normally implemented in web 
applications.
